# Legacy setting files

## 1. Lunarvim
- Lunarvim : https://www.lunarvim.org

### config file setting
```shell
$ cp lvim/.config ~/.config/lvim 
```

### alias setting
```shell
$ vi ~/.zshrc

alias vi="lvim"
```

### Updating LunarVim
* Inside LunarVim __:LvimUpdate__
